Decide$ – Personal Finance Management Application

Overview

Decide$ is a modern desktop application built with JavaFX that helps users efficiently manage their personal finances.
Developed as part of the PIDEV – 3rd Year Engineering Program at Esprit School of Engineering (Academic Year 2025–2026), the application provides powerful tools for tracking expenses, managing savings goals, visualizing financial data, and receiving intelligent financial insights.

The project aims to deliver an intuitive and interactive desktop experience while applying software engineering best practices, including layered architecture, modular development, and clean UI design.

* Secure user authentication and account management

Architecture

The application follows a layered and modular architecture:

* UI Layer (JavaFX/FXML)
    Handles the graphical user interface and user interactions.
* Controller Layer
    Manages communication between the interface and business logic.
* Service Layer
    Contains the application’s business logic and financial processing.
* Data Access Layer
    Handles database operations and persistence.
* Database Layer
    Stores users, transactions, savings plans, and analytics data using MySQL.
