# Decide$ – Personal Finance Management Application

## Overview
This project was developed as part of the PIDEV – 3rd Year Engineering Program at **Esprit School of Engineering** (Academic Year 2025–2026).  
It is a full-stack web application that helps users manage their personal finances, track expenses/revenues, plan savings goals, and receive smart insights for better financial decisions.

## Features
- User authentication and profile management
- Expense and revenue tracking
- Savings goals and financial planning
- Dashboard with statistics and visual charts
- AI-powered financial insights and recommendations

## Tech Stack

### Frontend
- Twig
- HTML/CSS
- JavaScript
- Bootstrap

### Backend
- PHP
- Symfony
- Doctrine ORM
- MySQL

## Architecture
The project follows an MVC architecture using Symfony:
- **Controllers** handle HTTP requests and responses
- **Services** contain business logic and AI-related processing
- **Repositories** handle data access with Doctrine ORM
- **Entities** represent domain models and database mapping

## Getting Started
1. Clone the repository
2. Install dependencies: `composer install`
3. Configure environment variables in `.env`
4. Run database migrations
5. Start the Symfony server
