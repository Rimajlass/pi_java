Vosk model setup for Smart Voice Command Agent

1. Download a Vosk English model (recommended: vosk-model-small-en-us).
2. Unzip it.
3. Place the folder as one of:
   - models/vosk-model-small-en-us
   - src/main/resources/models/vosk-model-small-en-us

The app checks both paths. If the model is missing, voice shows a friendly message and typed commands still work.
